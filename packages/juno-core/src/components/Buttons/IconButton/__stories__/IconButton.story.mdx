## Variant, Tooltip and `stretchIcon`

We support different variant button, that can be use in different use case.

default support `RcTooltip`, can set `title` to show tooltip, also support `TooltipProps` for custom tooltip props.

if you want browser default title, set `useRcTooltip={false}`
