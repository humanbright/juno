## Tooltip on disabled item

When you want make disabled item with tooltip, you can set `ignorePointer={true}`
that will wrap your compoent with a `span` mask

> also if you want custom that wrapper props, you can use `maskProps` to set that props,

<Canvas>
  <Story id="ðŸš€-cleanup-components-tooltip--tooltip-with-disabled-children" />
</Canvas>

if you want to change that default `span`, you can set styled-components props `as="div"`

```tsx
<RcTooltip
  title="disabled item tooltip"
  ignorePointer
  maskProps={{ as: 'div' }}
>
  <RcButton disabled>disabled item also have Tooltip</RcButton>
</RcTooltip>
```
