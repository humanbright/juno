# GlobalScrollBarStyle

> `GlobalScrollBarStyle` overwrite browser scroll bar follow by `juno` style.

[Go to this canvas](?path=/story/ðŸ’…-style-system-utils-globalscrollbarstyle--global-scroll-bar-style-example)

```tsx
import { RcThemeProvider, RcGlobalScrollBarStyle } from '@ringcentral/juno';

const App = () => {
  return (
    <RcThemeProvider>
      <RcGlobalScrollBarStyle />
    </RcThemeProvider>
  );
};
```
