### Use Svg

using `@svgr/webpack` to load svg as component.

you can import from `'@ringcentral/juno/icons/*.svg`, using like below.

```tsx
import phoneSvg from '@ringcentral/juno-icons/icon-phone.svg';

const App = (props) => {
  return (
    <RcThemeProvider>
      <RcIcon symbol={phoneSvg} />
    </RcThemeProvider>
  );
};
```
