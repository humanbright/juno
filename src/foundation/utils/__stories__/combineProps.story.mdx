### `Const` combineClasses

▸ **combineClasses**‹**T**›(`baseClasses`: T, `newClasses`: T): _NonNullable‹T›_

Combine two classes

**`example`**

```ts
combineClasses(
{
root: 'a1',
input: 'a1',
un1: undefined,
un2: 'a4',
},
{
root: 'a1',
input: 'a2',
un1: 'a3',
un2: undefined,
un3: 'a5',
},
)
// output result:
{
root: 'a1 a1',
input: 'a1 a2',
un1: 'a3',
un2: 'a4',
un3: 'a5',
}
```

### `Const` combineProps

▸ **combineProps**‹**T**›(`baseProps`: T, `newProps`: T, `coverPropKeys?`: keyof T[]): _NonNullable‹T›_

Combine props object together

that will group className, classes together
function will auto combine together,
trigger order is `intoProps.someFn(e) => defaultProps.someFn(e)`

**`example`**

```tsx
combineProps(
{
children: <A />,
value: '1',
className: 'a1',
classes: {
root: 'root1'
},
count: 1,
size: 'large',
onClick: fn1
},
{
children: <B />,
value: '2',
className: 'a2',
classes: {
root: 'root2'
},
count: 2,
size: undefined,
onClick: fn2
},
)
// output result:
{
children: <B />,
value: '2',
className: 'a1 a2',
classes: {
root: 'root1 root2'
},
count: 2,
size: 'large',
onClick: (e) => {
fn2(e);
fn1(e);
}
}
```

**Type parameters:**

▪ **T**

**Parameters:**

| Name             | Type      | Description                                                                             |
| ---------------- | --------- | --------------------------------------------------------------------------------------- |
| `baseProps`      | T         | base props you want to combine together                                                 |
| `newProps`       | T         | want to combine in to props                                                             |
| `coverPropKeys?` | keyof T[] | some prop key you want to cover, not combine when type is function, ex. `ComponentType` |

**Returns:** _NonNullable‹T›_
